Compliance Matrix

A compliance matrix for the CERT-J standards model is available in the Microsoft Excel XML file standards_info/compliance/cert-j_compliance.xml, which may be found relative to the LDRA tool suite installation directory.

This is a snapshot of CERT rules from 2009 and does not have the same numbering as the current wiki or the book.
